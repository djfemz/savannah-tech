## Environment Variables

To run this project, you will need to add the following environment variables to your example.env file

`DATABASE_HOST`
`DATABASE_USERNAME`
`DATABASE_PASSWORD`
`DATABASE_NAME`
`DATABASE_PORT`
`REPO_OWNER`
`REPO_NAME`
